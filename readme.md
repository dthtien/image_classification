# Image Classification with retrain [Tensorflow](https://www.tensorflow.org/) model
- Installation:
```sh
git clone git@github.com:dthtien/image_classification.git

cd image_classification

pip install flask

pip install tensorflow # tensorflow-gpu

python3 main.py
# open localhost:5000
```
