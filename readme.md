# Image Processing

- Installation:
```sh
git clone git@github.com:dthtien/image_classification.git

cd image_classification

pip install flask

pip install tensorflow # tensorflow-gpu

pip install keras
```
- Classification
  ```
  python3 main.py
  # open localhost:5000
  ```

- Object detection

  ```
  python3 predicter.py
  ```
